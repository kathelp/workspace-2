# Handling Rules

## Sensitive Topics
Escalate or be very cautious with:
- billing and refunds
- account ownership
- privacy requests
- security incidents
- data deletion
- legal threats

## Response Rules
- Do not disclose internal-only details.
- Do not expose another customer's information.
- Do not claim an action was taken unless it was actually taken.
- If unsure, say you need to verify.
